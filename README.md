# ECONMAC Indonesia Macroeconomic Dashboard

This is the group repository for our Introduction to Macroeconomics assignment. The assignment asks us to build an Aggregate Demand and GDP Dashboard for Indonesia. This repository stores all the data and the Excel workbook our group will use to complete the project.

## What is GitHub

GitHub is a website for storing and sharing files. Think of it like Google Drive but built for organized project work. You do not need to know how to code to use this. You only need to download one file from this page.

## What is in this repository

This repository holds three things. The first is the master Excel workbook named ECONMAC_Indonesia_Dashboard.xlsx. This workbook already has all the raw data loaded, and all the required formulas built in across eight separate sheets. The second is a folder named data, which contains the raw economic data we pulled directly from the FRED database at the Federal Reserve Bank of St. Louis. The third is a folder named instructions, which contains the original assignment PDF from our professor along with a plain text version for easy reading.

## How to download the Excel file

Go to the top of this page and look for the file named ECONMAC_Indonesia_Dashboard.xlsx in the list of files. Click on it. A new page will open showing a preview of the file. On the right side of that page, find the button that says Download raw file. Click it. The file will download to your computer. Open it with Microsoft Excel.

## What the Excel file already has done for you

The workbook has eight sheets. The Raw Data sheet contains the quarterly FRED data for Indonesia from 2000 through 2026, covering real GDP, private consumption, government consumption, investment, exports, imports, and population. The Clean Data sheet links directly to the raw data with standardized labels. The Growth Rates sheet calculates year on year growth for every component. The Expenditure Shares sheet shows each component as a percentage of GDP. The Contributions sheet shows the percentage point contribution of each component to GDP growth. The Correlations sheet computes how closely each component moves with GDP. The Welfare Indicators sheet calculates real GDP per capita and real consumption per capita. The Dashboard sheet and the Notes sheet are blank and ready for our charts and written commentary.
